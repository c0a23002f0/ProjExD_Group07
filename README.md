# テーマ

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
* ローグライクゲーム

## ゲームの実装
### 共通基本機能
* 背景、主人公と、敵の描画

### 担当追加機能
* main関数(担当：高橋)
* オブジェクト同士の距離算出関数(担当：高橋)
* フィールド内判定関数(担当：高橋)
* 主人公
* 武器（自動）
* 武器（手動）
* 敵
* 中ボス
* ラスボス
